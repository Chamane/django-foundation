Django Foundation

Django Foundation est une structure de projet Django, basé sur Python 3 et Django 2. Il a été crée dans le souci de développement rapide qui se veut un ensemble de configurations, de fonctionnalités déjà implémentées comme l'authentification, des recettes pour la sécurité, le project layout, les dépendences et différentes recettes pour le dépoiement

DOKLA a utilisé comme fondation Django Foundation.
Vous pouvez aussi aller regarder le stack de dokla

HOA (Haiti Online Academy) utilise Django Foundation pour développer sa plateforme de e-learning
